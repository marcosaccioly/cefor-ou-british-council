# 10.2 · Conceitos

Nós **atômicos**: um conceito por nota, navegável por links. A matéria-prima que a seção b do formulário vai manejar com precisão.

- Pergunta da camada: *qual o conceito atômico?*
- Exemplos a criar: `TNE.md`, `microcredencial-empilhavel.md`, `stackability.md`, `equivalencia-de-credito.md`, `RPL.md`, `TNE-de-microcredencial.md`.

## Convenção

- Um conceito por arquivo, título = conceito.
- Linkar para a fonte em `/10.1` que o define e para conceitos relacionados (`[[wikilink]]`).
- Sem opinião — interpretação vai para `20-permanent-notes/`.
