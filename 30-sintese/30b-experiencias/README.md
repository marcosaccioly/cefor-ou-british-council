# 30b · Experiências

Reuniões com o IET, sprints de co-desenho, piloto: o que a **prática** revelou. Cada experiência registra os conceitos ativados e o que o encontro mostrou de novo.

- Pergunta da camada: *o que a prática revelou?*
- Exemplo: cada reunião com Leigh-Anne/Felipe vira uma experiência — conceitos ativados, o que o IET vê diferente, insights que voltam pro inbox.

## Convenção (sugerida)

- Um arquivo por reunião/sprint: `YYYY-MM-DD_reuniao-iet.md`.
- Registrar: o que foi ativado, o que mudou no entendimento, próximos passos → inbox (`/00`) ou catálogo (`/40`).

> Fluxo: experiência (`/30b`) → insight (`/00`) → Permanent Note (`/20`) → MOC (`/30a`) → texto da seção (`/40`).
