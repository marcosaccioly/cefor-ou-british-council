# Constraint — escopo "exploratory"

**Guarda central: exploratory ≠ rodar curso.**

O recurso é para **estruturar e desenhar** a parceria, não para executar um curso inteiro. Todo texto que vai pra fora (concept note, seções do formulário) deve descrever atividades de **desenho, mapeamento, co-criação, piloto pequeno e disseminação** — nunca a operação de uma oferta completa.

## Aplicar em

- Seção c (atividades): entregáveis tangíveis de desenho/estruturação, não de oferta.
- Seção b (núcleo de TNE): framework e mapeamento de reconhecimento, não credenciamento já rodando.
- Seção d (resultados): blueprint, framework, piloto + roadmap — saídas exploratórias.

## Sinal de alerta

Se o texto promete "formar N turmas", "certificar N alunos" ou "rodar a microcredencial", está fora do escopo exploratory. Reescrever como desenho + piloto + roteiro de escala.
